# RbxPI Install Manager

[![CodeFactor](https://www.codefactor.io/repository/github/rbxpi/rbxmanager/badge)](https://www.codefactor.io/repository/github/rbxpi/rbxmanager)

This is the source code for the RbxPI Install Manager app.

For information about how to use the RbxPI install manager, including troubleshooting steps, please refer to the documentation at [rbxpi.github.io/rbxpi-core/guide/quick-start/#roblox-install-manager](https://rbxpi.github.io/rbxpi-core/guide/quick-start/#roblox-install-manager).

## Installation

To install RbxPI Install Manager you must already have [Git](https://git-scm.com/) and [Python](https://www.python.org/) installed on your machine.

```sh
> git clone https://github.com/rbxpi/rbxmanager
> cd rbxmanager
> pip install -r requirements.txt
> python3 rbxmanager
```

When launching RbxPI Install Manager you must ensure that your configuration (software and hardware) is compatible with rbxmanager.


## Release Schedule

Production releases of RbxPI Install Manager are very rare, and major updates are also very rare because RbxPI Install Manager was designed to be sustainable over time without requiring modifications.

Since RbxPI Install Manager relies heavily on HTTPS requests to communicate with the GitHub API, modifying its interface without changing the entire underlying logic is difficult. This is why major updates are both infrequent and challenging.
